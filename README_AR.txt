ADCO Smart Daily Plan v13 - Android APK GitHub Build

هذه نسخة مشروع APK مبنية على آخر نسخة v13.

طريقة الرفع على GitHub:
1. اعمل Repository جديد باسم:
   ADCO-Smart-Daily-Plan-v13
2. فك ضغط هذا الملف.
3. افتح الفولدر الناتج.
4. ارفع كل محتويات الفولدر كما هي على GitHub.
   لازم يظهر في الصفحة الرئيسية:
   app
   .github
   build.gradle
   settings.gradle
   gradle.properties
5. اضغط Commit changes.
6. افتح Actions.
7. اختار Build ADCO v13 APK.
8. اضغط Run workflow.
9. بعد الانتهاء، نزّل Artifact باسم:
   ADCO-v13-Smart-Daily-Plan-APK
10. فك الضغط وستجد:
   app-debug.apk

ملاحظة:
لو رفعت الملفات من الموبايل قد تتفك الفولدرات غلط. استخدم اللاب واسحب الفولدرات كما هي.
